# MACEDON

MACEDON is a Unity prototype for a third-person atmospheric RPG about a Varangian-style warrior.

Current milestone: Week 2 v2 playable vertical slice.

## Current Demo Flow

- Player starts in a small prototype world.
- Player can move, sprint, strafe, backpedal, jump, and punch.
- Player talks to a villager to start a wolf quest.
- Wolf spawns in the clearing.
- Wolf roams, detects the player, howls, chases, attacks, and returns home if leashed.
- Player can defeat the wolf or die.
- Death and victory screens allow restarting the demo.

## Tech

- Unity 6
- Built-in Render Pipeline
- WebGL build target
- Git/GitHub workflow
- itch.io deployment
